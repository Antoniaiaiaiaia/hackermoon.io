# [Hackermoon](https://hackermoon.io) 资源汇总

###### tags: `Web3`

Hackermoon是面向 Web3/Dweb 开发者的精神营地，我们通过将开发者关心的信息进行整合，希望能够节省一些大家的时间。

目前Hackmoon聚合的资源类型主要是：可以面向token编程的活动信息、羊毛信息，本仓库中的资料整理自网络，也有一些来自社区开发者的推荐。希望大家撸的开心:-D
